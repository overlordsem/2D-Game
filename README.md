# 2D-Game
2D Parkour game genaamd Five Nights at Sigma Obby. Gemaakt voor 1e periode game dev.

* Featured een simple movement system
* Moving Objects
* Death & Respawn System
* Werkende Pause functie

<img width="1073" height="495" alt="Screenshot 2025-11-18 102301" src="https://github.com/user-attachments/assets/78e114a7-66d6-48b8-bfb3-b45b48a25d90" />

Dit was mijn allereerste game op Unity. Hiermee leerde ik over C# en over hoe ik Unity moest gebruiken voor coderen, sound importeren, etc.

* 10 Week Project
* 6 Levels, en een Main Menu met functioning buttons
* Vol met references

# Inspiratie
Ik had veel inspiratie gekregen van de platforming game Geometry Dash. De player icon is ook een Cube van de game zelf en het maakt gebruikt van muziek uit Geometry Dash.
Ook zaten er refrences in naar Ninja de populaire streamer, NickEh30 en Chainsaw Man.
